Is a hand-curated set of games for the Miyoo Mini (running Onion OS) or the Anbernic RG-35XX (running Garlic OS).  Optional preview images are included for each game and all necessary bios and configuration files are also included.  You should be able to drag-and-drop everything on an SD card with a freshly installed version of Onion or Garlic.  The games should work for other devices and custom firmware  however you may need to do some research on how to properly add them to your SD Card, and also which BIOS files you will need.    •  The set is 6gb and  c ontains 1,900 games for: Arcade, Neo Geo, Atari 2600, TurboGrafx-16, Game Boy, Game Boy Color, Game Boy Advance, NES, SNES, Game Gear, Master System, and Genesis    •  A 64gb expansion adds 10 Sega CD, 10 TurboGrafx-16 CD, and 100 PlayStation games and brings the total size to about 55gb    •  A 128gb expansion adds 15 more Sega CD, 15 more TurboGrafx-16 CD, and 100 more PlayStation games and brings the total size to about 95gb ** * Interactive Games List *** Check out this interactive list of games by reddit user lewdev:   https://lewdev.github.io/apps/tiny-best-set-list/ so cool!!! ***  Random Game Selector  *** Check out this tool that will select some random games for you to play from the set:  https://go.madsoxgames.com/selector/ ***  Alternative Sets / Add-Ons  *** For Tiny Best Set Go! •   For Onion make sure you install each system using the package manager!   (See FAQ below) 2 - download games + bios files • For 32gb or smaller cards : download tiny-best-set-go-games.zip •  For 64gb cards : download  tiny-best-set-go-games.zip +  tiny-best-set-go-expansion-64-games.zip •  For 128gb cards :  download  tiny-best-set-go-games.zip +  tiny-best-set-go-expansion-64-games.zip + tiny-best-set-go-expansion-128-games.zip 3 - download preview art (optional) •  For 32gb or smaller cards : download tiny-best-set-go-imgs-XXX.zip where XXX=garlic or onion  •  For 64gb cards: download   tiny-best-set-go-imgs-XXX.zip +  tiny-best-set-go-expansion-64-imgs-XXX.zip •  For 128gb cards :  download  tiny-best-set-go-imgs-XXX.zip +  tiny-best-set-go-expansion-64-imgs-XXX.zip +   tiny-best-set-go-expansion-128-imgs-XXX.zip The default preview art was generated using this method   Note: If you don't like the "stretched" art you can use a classic 3 image mix instead.         you can find the corresponding art in the "alternate preview images (3 image mix)" folder 4 - download arcade name fixes (optional) •  Download: tiny-best-set-go-arcade-names- XXX.zip where XXX=garlic or onion 5 - Extract all of the .zip files that you downloaded into the same directory   Note: You do not have to unzip the individual roms.  The emulators can play the zipped roms! 6 - You should now have a single "Roms" and a single "BIOS" folder; copy these into your SD-card root •  The contents of the "Roms" folder should be copied into the "Roms" folder of your SD card •  The contents of the "BIOS" folder should be copied into the "BIOS" folder of your SD card 7 - For Garlic, you will also have a "CFW" folder; copy this into the root of your SD1 card •  The contents of the "CFW" folder should be copied into the "CFW" folder of your SD card •  If you have a 2 card installation the "CFW" folder should be on the first card If you plan on playing a lot of arcade games, you may wan to see the Arcade Update pack that fixes a few issues. * FAQ / Notes * Onion Packages Required After doing a fresh install of Onion, make sure the following packages are installed using the package manager: • Arcade • Atari - 2600 • NEC - TurboGrafx CD   (for 64 and 128 expansions only) • NEC - TurboGrafx-16 • Nintendo - Game Boy • Nintendo - Game Boy Advance • Nintendo - Game Boy Color • Nintendo - NES • Nintendo - SNES • SNK - Neo Geo • Sega - CD  (for 64 and 128 expansions only) • Sega - Game Gear • Sega - Genesis • Sega - Master System • Sony - Playstation  (for 64 and 128 expansions only) Arcade Name Fixes Both Garlic and Onion use different internal databases to generate Arcade names in the game list menu.  Those databases do not have information for  all of the Arcade roms provided in this set. This is why additional configuration files have been provided to fix the arcade display names, should you choose to use them.   For Garlic, an updated configuration file is deployed to the CFW folder that includes names for the missing games.   For Onion, a miyoogameslist.xml file is deployed as this is the only way to adjust arcade names.   Preview Images There are separate zip files for Garlic and Onion preview images because artwork specifications are different for each firmware. All preview images were scraped using this guide . Note that for Garlic, the images pack will also install a configuration setting that modifies the alignment and margins of the game menu text as described in the guide.  If you install the images and then want to use a different set of images you may have to re-adjust the configuration file on your SD-1 card which is in CFW-&gt;Skin-&gt;settings.json Adding Games If you want to add more games you should simply add them to the corresponding folder on your SD card (see Onion or Garlic documentation for more details).  Note that for Onion you will also have to " refresh your roms list " by pressing the menu button and then "refresh roms" from the main menu.   If you also want preview images you will have to generate them using this guide .  This should work for all systems except Arcade and PlayStation (see below...) Adding Arcade Games To add more arcade games make sure you are adding Full-Non Merged roms from a Mame 2003 plus set.   Note that for Onion, if you install the "Arcade Name Fixes" you will have to re-scrape the arcade folder to generate a new miyoogamelist.xml before your game will show up... or you will have to modify the miyoogameslist.xml file in the Arcade folder and make an entry for your new game(s).   If you want to mess with Arcade stuff, best of luck to you =) Adding PlayStation Games It is recommended to use the .chd format for PlayStation games.  For single disc games simply add the new .chd file to the PS folder.  For multi-disc games you should put all .chd files into PS/_hidden/multi-disc and then in the PS folder create a .m3u file for the game.  You can see the .m3u files in the current set and use them as guides.  This is to ensure multi-disc PlayStation games only show up once.  the multi-disc directory is nested 2 folders deep so that it does not show up in the game list (games placed in single nested folder will show up, as will the folder itself) Game Changes From Original Tiny Best Set No games were removed since the original Tiny Best Set, but the following changes have been made: -  Since many translated games had file names that were too long for the Onion Menu,  All translation suffixes have been removed and replaced with "(Translation)" - Since the games will appear in the menu using their file names instead of a gameslist.xml file,  a handful of games had their filename modified: Genesis: Bare Knuckle - Ikari no Tekken ~ Streets of Rage (World) (Rev A) =&gt;
