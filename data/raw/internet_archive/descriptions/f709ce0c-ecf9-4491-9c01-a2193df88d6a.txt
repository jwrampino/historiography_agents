These are ISO images created with ImgBurn from clean Windows 7 Professional SP1 install disks (32 bit and 64 bit respectively). These ISOs are English by default. Hopefully these are of use to someone who wishes to create virtual machines, or even install on older hardware! The 3.1 GB file is 64 bit, while the 2.4 GB file is 32 bit. This can be verified by the folder structure:  https://archive.org/download/win-7-pro-32-64-iso
