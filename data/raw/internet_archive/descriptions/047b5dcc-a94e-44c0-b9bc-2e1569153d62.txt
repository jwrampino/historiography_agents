Arch Linux (or Arch /ɑːrtʃ/) is a Linux distribution for computers based on x86-64 architectures. This collection contains historical versions of all Arch Linux packages starting from September 2013.&nbsp; It includes both x86_64 and i686 packages, up until November 2017 where support for i686 was dropped in Arch Linux. This complements the Arch Linux Archive which only keeps old packages for a few years.
