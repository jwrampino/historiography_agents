Hearto's 1g1r collection. This is a collection of many systems in the 1g1r format using the best possible trimmer tool on clean dumps for archival purposes. When possible (90% of the sets) retool was used to trim the romsets , for the same reason this set has preference for the latest and best version of each game. The rules used was: No bad dumps, no hacks, no fixed  or cracked roms, only games related content. For more modern consoles, I prefer to wait more years to be sure they are abandonware. Can you update the set? Sadly it is very hard to upload big files in one go to this site and the torrent files are generated from the first upload. This could lead in broken torrents after updating files. Direct downloads from archive.org doesn't have this problem and are always updated. I have uploaded this torrent in one go (2 days uploading the file) so the torrent is complete, and an update could break it.  I notice some weird decisions though like using some Sonic roms from the DS game Sonic Classic Collection instead of the originals? Retool was used for the filters to remove duplicates. For example if you have Chrono Trigger Jap, Us, Us1.1 and Eur, retool leaves only the US 1.1 version.
