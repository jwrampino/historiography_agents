A collection of DOS games from Major Geeks website. Description from the site itself: Included are 524 different games for DOS from the 1980s which will take you on quests, battles, and a journey. This is a great collection but playing old DOS games is not all that simple if you've never seen DOS before. Since you have 524 games, each in its own zip file and using long filenames, you will want to use some special tools and spend some time getting everything organized. We would suggest using DOSBox as a DOS emulator alongside D-Fend Reloaded as a GUI for DOSBox (again if you can't use DOS). Drop To Dos is neat for advanced DOS users.
