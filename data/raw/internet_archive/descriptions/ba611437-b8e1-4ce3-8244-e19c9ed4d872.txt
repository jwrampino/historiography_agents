This page is one of several which provide a collection of plain vanilla CD case inserts for LibriVox audio books. Each insert is intended to be placed into an empty CD case along with the corresponding audio book, on CD or DVD. These inserts list the meta-data related to a LibriVox audio book. This meta-data includes items like: the book title, authors name, run time, number of chapters, description of the audio book, reader and support staff credits and so on. Each insert included in the collection corresponds to a particular LibriVox audio book. The inserts here are in PDF files. The insert should be downloaded, printed, then cut from the sheet and inserted into an empty CD case, along with the CD/DVD containing the audio book. The inserts in this collection for only a subset of the all the available LibriVox audio books. The inserts on each Web page are sorted by the audio book name (rather than author). However, there is no particular order of inserts across the several Web pages of this collection. ##### There are two types of inserts in the collection. The first insert type is for a "standard" or fat CD case. The second insert type is for a "slim" CD case. There are no inserts here for the large DVD cases. The standard/fat CD case is a three piece case with a cover, holder and tray. This case allows for a two individual inserts, a cover insert and a tray insert. The cover insert includes the book title, author name and a listing of the chapter details. If space permits the detailed chapter titles and readers is included. The cover insert is placed under the tabs in the cover. The tray insert includes the audio book description from the LibriVox web page, author name, total run time and chapter/audio segment description. The tray insert is sandwiched between the holder and tray pieces. The slim CD case is a two piece case with a cover and combined holder/tray. This case allows for only a cover insert. The cover insert lists the audio book name, authors name, the audio book description from the LibriVox web page, total run time and chapter/audio segment description. ##### In addition, there are multiple-CD case "extras" for some audio books. These "extras" are inserts for multiple (2 to 6) CD cases, intended for audio books that are in a series, or are related (as in a book and its sequel, etc.). For example, a 2-CD case insert is available for "Relativity" and "Sidelights on Relativity", or a 4-CD case insert is available for the four Sherlock Holmes novels. There are others, including multiple-CD case inserts for works of Edgar Rice Burroughs, H. Rider Haggard and Shakespeare. Furthermore, there are image "extras" for some audio books. These "extras" are PDF files that include pictures, maps or figures pertaining to an audio book. These images have been collected from the e-text corresponding to the audio book. These "extras" are provided in order to aid in the enjoyment of the audio book. ##### The Web pages for other volumes are at Plain Vanilla CD Case Inserts: Volume 1 , Volume 2 , Volume 3 .
