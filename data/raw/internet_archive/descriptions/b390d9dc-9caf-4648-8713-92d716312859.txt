Note to get PVR client to work do the following steps. 1. Navigate to "System" tab. 2. Select Live TV. 3. In the General tab of Live TV settings click on "Enable". 4. It will ask to select a PRV Client, please select "IPTV Simple Client". 5. Select the "Enable" option and press back "twice" so that your back on the Live TV General options. 6. Now you must select "Disable". 7. Then you must select "Enable" again from the same tab. 8. It will ask you to select your client again. 9. Like before select "IPTV Simple Client" and then select "Enable" 10. Now you should see your PRV being updated. ################ Library Instructions: If a user wishes to add their personal libraries, add them to the directory: userdata/library/ Please check favourites menu for options to search, clean, browse, export, and update libraries.
