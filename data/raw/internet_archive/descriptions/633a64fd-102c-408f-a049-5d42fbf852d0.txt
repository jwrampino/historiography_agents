A collection of high-quality preservation copies of&nbsp; vintage software, with an emphasis on NEC PC-9800 and PC-8800 series computers .scp = multi-revolution flux image, for preservation and converting to other formats .mfi = single-revolution flux image, for using copy-protected software on MAME .hfe = single-revolution bitstream image, for writing new floppies and HxC/Gotek use .d88 = emulator-friendly image, cracked (when needed) using OG Japanese "personal backup" software (aka "filers") A key feature of these dumps is that copy-protection schemes are preserved. To run them on emulators, use the cracked .d88 images provided. With MAME specifically, you can also use the protected .mfi images, but note that there are still issues with some protection types.
